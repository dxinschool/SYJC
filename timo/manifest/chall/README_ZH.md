# 選手須知

## 背景

“Manifest”是一套港口調度終端的離線版本，提供登錄、名冊登記、清單維護、貨箱導入、事故上報與封存診斷等功能。

## 附件

- `/home/ctf/awd`

## Patch 說明

1. 只允許修改漏洞點附近的 `.text` 段。
2. 總改動字節數不得超過 6 字節。
3. 文件大小、段佈局必須保持一致，不允許新增段或更換連結方式。
4. 運行環境為受限用戶與沙箱，禁止新增外部依賴或系統調用。
5. 提交頻率限制：每5分鐘最多2次。

## Patch 提交流程

1. 將補丁後的 ELF 放到 `/tmp/`。
2. 執行：

```
sudo /usr/local/bin/check /tmp/your_elf
```

- 通過後會自動替換為 `/home/ctf/awd`。
- 未通過只返回失敗資訊，不提供具體原因。
